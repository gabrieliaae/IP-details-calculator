This app will help you get details about the IP you enter in the input.For it to run you will need to download the files and an external code editor. I used Visual Studio Code. You can download it through the link:
https://code.visualstudio.com/
After installing it you will also need to download the Live Server extension and run index.html to review the projects.
Here is a small review of what each project does:

It can calculate:

Subnat mask, WildMask, Network, BroadCast, FirstHost Address, Lasthost adress, HostRange.
